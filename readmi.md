\# Website Company Profile - Rosoft Digitalindo



Website company profile profesional untuk Rosoft Digitalindo, sebuah perusahaan IT Solution \& Development yang menyediakan layanan teknologi dan solusi perangkat lunak untuk bisnis.

\## Fitur Utama



1\. \*\*Halaman Beranda\*\* dengan hero section yang menarik

2\. \*\*Carousel Cinematic\*\* dengan 3 gambar showcase

3\. \*\*Section Tentang Kami\*\* dengan visi dan misi

4\. \*\*Layanan\*\* (6 kategori solusi IT)

5\. \*\*Portofolio\*\* dengan contoh proyek

6\. \*\*Tim\*\* dengan profil anggota

7\. \*\*Kontak\*\* dengan form dan informasi kontak

8\. \*\*Footer\*\* lengkap dengan tautan sosial media



\## Teknologi yang Digunakan



\- HTML5

\- CSS3 dengan custom properties (variables)

\- JavaScript (ES6+)

\- Bootstrap 5 (CDN)

\- Font Awesome (CDN)

\- Google Fonts (Poppins)



\## Cara Menjalankan



1\. Clone atau download proyek ini

2\. Buka file `index.html` di browser web

3\. Website sudah responsif dan siap digunakan



\## Cara Menyesuaikan Konten



\### Mengubah Konten Halaman

\- Buka file `index.html` untuk mengubah teks, gambar, dan struktur

\- Sesuaikan URL gambar dengan gambar yang Anda miliki



\### Mengubah Tampilan

\- Buka file `style.css` untuk mengubah warna, font, dan layout

\- Variabel CSS utama ada di bagian atas file



\### Mengubah Fungsi

\- Buka file `script.js` untuk mengubah atau menambahkan fungsi JavaScript



\### Informasi Kontak

\- Ubah informasi kontak di bagian Contact Section dan Footer

\- Ganti nomor WhatsApp di tombol Chat Sekarang



\## Responsif

\- Desktop (≥992px)

\- Tablet (≥768px)

\- Mobile (<768px)
